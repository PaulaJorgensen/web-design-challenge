Summary: Latitude vs X.

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs latitude. Factors looked at included: temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as, explaniations and descriptions of any trends and correlations witnessed.
